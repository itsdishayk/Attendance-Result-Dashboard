## What's Inside This Project?

This dashboard is divided into key modules to help students/teachers manage data effectively:

### 1. **Attendance Tracking**
* Used structured forms to input student names and attendance status.
* Integrated `date` inputs to track attendance for specific days.

### 2. **Result Management**
* Created a clean layout to display marks and grades.
* Used semantic `<section>` and `<article>` tags for better accessibility.

### 3. **The Tech Stack**
* **HTML5:** Semantic structure, Forms, and Multimedia.
* **Git:** Version control to track code changes.
* **GitHub:** Hosting the code in a remote repository.
* **GitHub Pages:** Deploying the project live for the world to see!

##  Key Learnings from Day 5

* **Git Workflow:** Learned how to `add`, `commit`, and `push` code via the terminal.
* **Deployment:** Understanding how to host a static website for free.
